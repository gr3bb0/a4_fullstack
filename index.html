FULL/STACK/LAMP

what does 'full' in full stack mean?

full means that you develop both the front end and back end. what you see when you enter a website on the clientside is frontend, and looks pretty. backend is all the stuff that makes it work.

what does 'stack' in full stack mean?

stack is all the programs and tools used in a website. for example, frontend is html5, css3, and js, while backend is php and mysql. i also have a lamp stack on my server.

LAMP stack? what do lamps have to do with this?

lamp is a stack in the server, which makes it so you can see the webpage. lamp stands for: 
			
Linux: the operating system we use on the server
Apache: the webserver itself, where everything is housed
MySQL: the database we use for information
PHP: programming language used serverside
